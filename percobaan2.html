fyi:span adalah salah satu tag/elemen yang berfungsi untuk menyisipkan sesuatu yang khusus.kebanyakan orang biasanya menggunakan tag span untuk melakukan styling dengan menggunakan css.kebanyakan orang sering bingung apa perbedaan dari span dan division,keduanya memamg memiliki fungsi yang sama,yaitu menghubungkan informasi ke potongan kelompok kode html,paling sering dengan menggunakan atribut class dan id untuk mengasosiasi elemen dengan atribut class atau id dari css pemilih.

perbedaan antara tag division dan tag span adalah pada jenis tagnya.tag division adalah tag block line sementara tag span adalah tag in-line,bagi teman teman yang belum tahu apa itu block line dan in-line,saya akan membuatkan artikel lainnya yang akan membahas khusus tentang apa itu tag block line dan apa itu tag in-line
